# Netflix Shows Exploratory Data Analysis

## Overview

This project presents an exploratory data analysis (EDA) of Netflix shows and movies. The goal was to investigate the characteristics of the Netflix content catalog, identify patterns in ratings, genres, release years, and audience categories, and derive insights that could support content-related business decisions.

The project was completed as part of the Data Analysis and Machine Learning course at HSE University.

---

## Objectives

* Explore the structure and quality of the dataset.
* Analyze content distribution across ratings and genres.
* Investigate trends in release years and audience targeting.
* Identify relationships between content characteristics and user ratings.
* Generate business-oriented insights through data visualization.

---

## Dataset

The dataset contains information about Netflix content, including:

* Title
* Rating category
* Rating level description
* Release year
* Genre
* User rating score
* Audience information

---

## Analysis Workflow

### 1. Data Preparation

* Loaded and inspected the dataset.
* Removed irrelevant features.
* Checked for missing values and inconsistencies.
* Performed basic data cleaning and preprocessing.

### 2. Exploratory Data Analysis

* Analyzed distributions of ratings and genres.
* Investigated content release trends over time.
* Examined user rating patterns.
* Compared characteristics of different audience categories.
* Identified outliers and unusual observations.

### 3. Data Visualization

Built visualizations to explore:

* Content distribution by rating category.
* Genre popularity.
* Release year dynamics.
* User rating distributions.
* Relationships between key variables.

### 4. Business Insights

The analysis was used to:

* Understand how Netflix content is distributed across audience segments.
* Identify the most common content categories.
* Explore trends in content production over time.
* Highlight factors associated with higher user ratings.
